Información general

Sobre la aplicación

Proyecto Notas es una aplicación web construida con el framework Django que permite crear, editar y organizar notas de manera rápida y centralizada.

Datos del proyecto

  • Autor: Ignacio Guzmán
  • Carrera: Técnico en Programación – UTN
  • Actualmente en: FASE – Soporte Técnico y Sistemas
  • Tipo de proyecto: Práctica de desarrollo con Django

Tecnologías utilizadas

  • Django: framework principal para la lógica del lado del servidor.
  • Templates HTML: sistema de plantillas de Django para las vistas.
  • Bootstrap 5: maquetado responsivo y componentes visuales.
  • Base de datos: SQLite (modo desarrollo).

Objetivos de aprendizaje

  • Practicar la estructura de proyectos y aplicaciones en Django.
  • Trabajar con vistas, URLs y templates reutilizables.
  • Diseñar una interfaz limpia usando Bootstrap.
  • Implementar operaciones básicas de CRUD sobre notas.

Próximas mejoras

  • Agregar búsqueda y filtros de notas.
  • Incorporar categorías o etiquetas.
  • Guardar la fecha de creación y última modificación.
  • Agregar autenticación de usuarios para notas personales.
